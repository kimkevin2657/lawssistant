<기획전 관리>

① [선택수정] : 기획전명을 수정하고 체크박스 체크 후 선택수정이 가능합니다..
② [선택삭제] : 체크박스 체크 후 버튼을 클릭시 기호기전을 삭제 합니다.
③ [기획전등록] : 신규 기획전을 등록합니다.

<기획전 등록/수정>

① [기획전명] : 기획전명을 입력합니다.
② [기획전URL] : 기획전 등록시 베너링크가 자동으로 생성됩니다. 기획전바로가기버튼을 클릭시 해당 기획전페이지로 이동합니다.
③ [관련상품코드] : 해당 기획전에 맞는 상품코드를 1개씩 세로로 추가합니다.
④ [목록이미지] : 기획전 노출 전체페이지이나 복지몰 제휴서비스로연결되어 사용불가.
⑤ [상단이미지] : 기획전 상단에 이미지가 노출됩니다. 가로 1200px * 세로
⑥ [저장] : 해당 내용을 등록 및 수정하여 저장버튼을 눌러줍니다.
