<공급사 정산관리>

① 주문금액 : 판매가 + 옵션가 (배송비 제외!)
② 쇼핑포인트결제 : 업체 주문건중 소비자회원이 주문시 사용 결제한 쇼핑포인트금액입니다.
③ 쿠폰할인 : 업체 주문건중 쿠폰사용으로 할인 된 금액 입니다.
④ 배송비 : 업체 주문건중 발생 된 배송비 금액 입니다.
⑤ 공급가총액 : 업체 상품공급가 (배송비 제외!)
⑥ 실정산액 : 공급가총액 + 배송비
⑦ 가맹점수수료 : 현 업체 주문건중 지급된 가맹점 수수료 금액입니다. 분양몰도 본사상품 기준이므로 (현재사용불가)
⑧ 본사마진 계산법
　 a.결제수수료가 본사 부담 :(주문금액 - 공급가총액 - 배송비 - 쿠폰할인 - 쇼핑포인트결재) = 본사마진)
⑨ "선택정산" 버튼을 클릭하시면 해당업체별 정산완료가 가능합니다.
⑩ 건별정산내역"내역" 버튼을 클릭하시면 해당업체의 건별 상세정산내역 가능하며 건별 정산처리가가능합니다. 단, 선택정산완료 전에만 사용이가능합니다.
