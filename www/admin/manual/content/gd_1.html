<전체 상품관리>

① [상품수정] : 버튼을 클릭시 "상품수정" 모드로 접속 합니다.
② [선택삭제] : 체크박스 체크 후 버튼을 클릭시 개별적으로 상품을 삭제 합니다.
③ [선택순위수정] : 순위를 변경 후 체크박스 체크 후 "선택순위수정" 버튼을 클릭합니다.
④ [선택상품복사] : 복사 하실 상품을 체크박스 체크 후 "선택상품복사" 버튼을 클릭하시면 동일한 상품이 복사됩니다.
⑤ [+상품등록] : 신규상품을 등록 합니다.
⑥ [엑셀저장] : 상품정보를 엑셀(CSV)로 저장 합니다. 엑셀저장은 "상품 일괄수정"에서도 사용 됩니다. 매뉴얼 > 상품 일괄수정 참조!
