<주문관리 공통>
  • [오늘 접수된주문] : 금일 주문된 내역만 노출 됩니다.
  • [1단계_입금대기] : 무통장입금 및 가상계좌로 결제시 노출 됩니다. (무통장입금,가상계좌 주문건)
  • [2단계_입금완료] : 결제완료된 주문건이 노출 됩니다. 신용카드,실시간계좌이체
  • [3단계_배송준비] : 주문상품이 출고 전 상태입니다.
  • [4단계_배송중] : 주문상품을 출고 후 상태입니다.
  • [5단계_배송완료] : 배송이 완료 된 주문상품이 노출 됩니다.
  • [배송전 환불] : 신용카드,계좌이체,핸드폰결제 후 취소한 내역이 노출 됩니다.
  • [입금전_주문취소] : 무통장입금 및 가상계좌로 주문결제 후 취소한 내역이 노출 됩니다.
  • [배송후 반품] : 배송이 완료된 주문건중 반품처리된 내역이 노출 됩니다.
  • [배송후 교환] : 배송이 완료된 주문건중 교환처리된 내역이 노출 됩니다.
  • [일련번호] : 상품주문시 개별적으로 중복되지않는 고유값 입니다.
  • [주문번호] : 여러상품을 복수구매를 하더라도 주문번호는 동일하게 적용됩니다.
  • [회원] : 회원이 주문시 주문내역관리에 노출 됩니다.
  • 　  PC버전에서 주문시 주문번호가 나옵니다.
  • 　  모바일쇼핑몰에서 주문시 주문번호(M) 나옵니다.
  • [결제방법] : 신용카드,무통장,계좌이체,핸드폰,에스크로 계좌이체중 하나가 노출 됩니다.
  • [판매자] : 실제 상품을 발주할 회원정보입니다.
